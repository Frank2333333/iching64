# IChing64 - 易经六十四卦学习平台

## Try Me
https://www.iching64.fun/

## 核心功能

- ✨ **六十四卦完整数据**: 包含所有卦象的卦辞、彖传、大象传、爻辞、小象传
- 🎨 **交互式卦象展示**: 可视化卦画、阴阳爻、上下卦结构
- 🔍 **智能搜索**: 支持按卦名、拼音、中文名搜索
- 🔄 **卦变关系**: 展示对卦(错卦)、综卦(覆卦)、互卦等关系
- 🌟 **五行属性**: 每个卦象都有对应的五行属性展示
- 📱 **响应式设计**: 支持桌面端和移动端访问
- 🌙 **深色模式**: 墨黑金韵配色，护眼阅读
- 🔢 **数字起卦**: 支持数字起卦法，输入三位数字即可起卦
- ✨ **流畅动画**: 卡片交错动画、页面切换动画、波纹点击效果
- 🎋 **传统纹样**: 祥云背景纹样，营造传统文化氛围

## 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS + shadcn/ui
- **图标库**: Lucide React
- **状态管理**: React Hooks
- **路由**: React Router

## 快速开始

### 前置要求

- Node.js 20+
- npm 或 pnpm

### 安装依赖

```bash
npm install
```

### 开发服务器

```bash
npm run dev
```

启动本地开发服务器，默认地址: http://localhost:5173

```bash
npm run build
```

生成优化后的生产环境构建文件到 `dist/` 目录

### 预览生产构建

```bash
npm run preview
```

### 代码检查

```bash
npm run lint
```

## 使用说明

### 浏览卦象

1. 打开应用后，可以看到所有六十四卦的卡片展示
2. 每个卡片显示卦序号、五行标识、卦画、卦名和上下卦信息
3. 点击任意卦象卡片进入详情页面

### 查看卦象详情

在详情页面可以查看：
- 卦象基本信息（卦画、卦名、五行、上下卦）
- 卦辞、彖传、大象传
- 六爻详情（爻辞、小象传、当位情况）
- 时位属性分析
- 卦变关系（对卦、综卦、互卦）
- 卦变列表
- 点击相关卦象可跳转到对应详情页

### 搜索功能

在顶部搜索框中输入：
- 卦名（如："乾为天"）
- 中文名（如："乾"）
- 拼音（如："qian"）

### 数字起卦

1. 点击导航栏的「数字起卦」按钮
2. 输入三个三位数字（或点击「随机生成」）
3. 系统自动计算下卦、上卦、动爻
4. 查看卦象结果和动爻详解

### 主题切换

点击导航栏的 ☀️/🌙 按钮可切换浅色/深色模式：
- **浅色模式**: 琥珀暖色调，清新明亮
- **深色模式**: 墨黑金韵，沉稳大气，适合夜间阅读

## 样式主题

### 浅色模式
- **主色调**: 琥珀色系（amber）- 体现传统文化特色
- **背景**: 渐变暖色调（amber-50, orange-50, yellow-50）

### 深色模式（墨黑金韵）
- **背景**: 墨黑色（#0a0a0a）
- **卡片**: 深灰黑（#171717）
- **强调色**: 金色（#d4af37）
- **边框**: 金色微光边框

### 五行颜色
- 金: #FFD700 (金色)
- 木: #228B22 (森林绿)
- 水: #1E90FF (道奇蓝)
- 火: #FF4500 (橙红)
- 土: #8B4513 (马鞍棕)

## 特性详解

### 动画效果
- **卡片加载**: 64个卡片依次淡入，错落有致
- **点击反馈**: 波纹扩散效果
- **页面切换**: 滑入滑出过渡
- **滚动触发**: 详情页各区块渐入动画

### 滚动位置记忆
- 自动记住列表页和详情页的滚动位置
- 返回时恢复之前的阅读位置

### 传统纹样
- 祥云图案背景，淡淡飘动
- 深色模式下呈金色，浅色模式下呈琥珀色
