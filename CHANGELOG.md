## [Unreleased]

### Added - 新增功能

- **深色模式** - 新增墨黑金韵深色主题
  - 支持浅色/深色模式切换
  - 主题偏好自动保存到 localStorage
  - 自动检测系统主题偏好
  - 金色卦画、祥云纹样适配深色模式

- **数字起卦功能** - 新增数字起卦页面
  - 支持输入三位数字进行起卦
  - 随机数字生成功能
  - 展示起卦计算过程
  - 动爻高亮显示
  - 查看完整卦象详情

- **动画效果** - 全面提升交互体验
  - 卡片加载交错淡入动画（64个卡片依次出现）
  - 卡片点击波纹扩散效果
  - 页面切换滑入/淡出动画
  - 详情页各区块滚动触发淡入
  - 返回按钮悬停滑动效果
  - 六爻依次显示动画
  - 卦画绘制动画

- **滚动位置记忆** - 优化浏览体验
  - 自动记住列表页和详情页滚动位置
  - 返回时自动恢复阅读位置
  - 防抖优化性能

- **传统纹样背景** - 营造传统文化氛围
  - 祥云纹样背景，缓慢飘动
  - 回纹图案装饰
  - 深色模式下呈金色，浅色模式下呈琥珀色
  - 使用 CSS 动画实现流畅效果

- **主题切换按钮** - 新增主题切换组件
  - 太阳/月亮图标切换动画
  - 旋转过渡效果
  - 悬停光效反馈

### Changed - 变更

- **深色模式配色** - 全新墨黑金韵配色方案
  - 背景色：墨黑 (#0a0a0a)
  - 卡片色：深灰黑 (#171717)
  - 强调色：金色 (#d4af37)
  - 文字色：米白 + 金色高亮
  - 边框：金色微光效果

- **按钮样式** - 深色模式按钮改为金色
  - 数字起卦按钮深色模式金色背景
  - 导航按钮深色模式金色高亮

- **项目结构** - 优化代码组织
  - 新增 `hooks/` 目录存放自定义 Hooks
  - 新增 `pages/` 目录存放页面组件
  - 新增 `components/ThemeToggle.tsx` 主题切换组件

### Fixed - 修复

- 修复主题切换按钮点击无响应的问题
- 修复深色模式下部分文字对比度不足的问题
- 修复卡片悬停效果在深色模式下的显示

## [0.0.0] - 2026-02-26

### Added - 初始版本

- **六十四卦完整数据**
  - 包含所有卦象的卦辞、彖传、大象传
  - 六爻详情（爻辞、小象传、当位情况）
  - 时位属性分析
  - 卦变关系（对卦、综卦、互卦、卦变列表）
  - 五行属性标注

- **交互式卦象展示**
  - 可视化卦画展示（阴阳爻）
  - 上下卦结构展示
  - 五行颜色标识

- **智能搜索**
  - 支持按卦名搜索
  - 支持按中文名搜索
  - 支持按拼音搜索

- **响应式设计**
  - 桌面端布局（8列卡片网格）
  - 平板布局（4-6列自适应）
  - 移动端布局（2列卡片）
  - 移动端侧边栏菜单

- **技术栈**
  - React 19 + TypeScript
  - Vite 构建工具
  - Tailwind CSS + shadcn/ui
  - React Router 路由
  - Lucide React 图标

## 版本说明

### 版本号格式
`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 分类标签

- `Added` - 新增功能
- `Changed` - 变更（对现有功能的更改）
- `Deprecated` - 弃用（即将移除的功能）
- `Removed` - 移除（已经移除的功能）
- `Fixed` - 修复（bug 修复）
- `Security` - 安全（安全性改进）
